# Capstone_3 : Analysis with Power BI

## Overview
For this Capstone project it will be the analysis of four years of sales data for EmporiUm, the team must create a Power BI report that highlights key insights, including trends over time and insights into recent sales highlights, to be shared with the regional sales team as well as territory managers and individual store managers.

## Tools Used
- Power BI
- Excel
- GitHub
- Power Query
