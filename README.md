# Capstone_3 : Analysis with Power BI

## Overview
For this Capstone project it will be the analysis of four years(2022-2025) of sales data for EmporiUm, the team must create a Power BI report that highlights key insights, including trends over time and insights into recent sales highlights, to be shared with the regional sales team as well as territory managers and individual store managers.

## Tools Used
- Power BI
- Excel
- GitHub
- Power Query
- DAX measures

## Sales Overview
- Monthly sales trends
- Store and online sales comparison
- Revenue share by state

## Data cleaning
- Fixed a missing category code
- Fixed inconsistent category naming across tables
- Built connecting Table Relationships

## Findings
- Overall sale increase between 2022 and 2025
- NY store brought in more revenue than CT
- Technology & Accesories product generated the highest revenue
